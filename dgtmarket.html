DGTMARKET

Landing Page Redesign

Challenges

dgtmarket.com is a modern exchange platform that allows to buy and sell Bitcoin and Litecoin. Their landing page needed refreshing - the idea was to design it in a more engaging interactive way and present the new content. The main impact was put on graphic solutions and content structure.

Solutions

  • keep the consistency with the existing logo and key visuals
  • design custom illustrations referring to the current trends in graphic design
  • add animations and visual effects to involve the user and catch his attention
  • take care of Responsive Web Design
  • divide the content into independent sections to make it more readable
  • focus on Call To Action

Tools used

Affinity Designer / webpack2 / sass / Bootstrap / ScrollMagic.js / svg animations / Bitbucket / sketchbook
