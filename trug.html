Challenges

TRUG stands for Tricity Ruby User Group - it's the group of people that meet every month in Sopot to discuss about programming (mainly in Ruby language, but not only). The task was to redesign the existing logo and website to fit current standards & design trends.

Solutions

  • logo referring to the anchor symbol, Gdańsk architecture reflecting on the water and the ruby sign connected with the programming language
  • make the website responsive
  • custom hero graphic with the city panorama containing architectural elements of Gdańsk, Sopot & Gdynia
  • care about the consistency in style (colors, rounded edges, linear graphics & icons)
  • add CTA on the top in case the user wants to see directly when & where is the next meeting
  • put an impact on social aspects

Tools used

Affity Designer / Affinity Photo / sketchbook / Ruby Sinatra / Slim / Sass/ hocus-pocus CSS Framework / GIT
